# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in uniple checkout for Shopify,
please email support@uniple.io with details. Do not file a public GitHub issue
for security reports.

We will acknowledge your report within 5 business days and provide a
remediation timeline based on the severity.

Please include:

- A summary of the vulnerability
- Affected versions and merchant impact
- Reproduction steps, if available
- Your preferred contact address

## Remediation Targets

These are target timelines after we confirm the issue and severity.

| Severity | Target |
|---|---|
| Critical | Mitigation within 3 business days, fix target within 7 business days |
| High | Fix target within 14 business days |
| Medium | Fix target within 30 business days |
| Low | Next planned maintenance release |

## Supported Versions

| Version | Supported |
|---|---|
| 0.1.0 | ✓ |
| < 0.1.0 | ✗ (please upgrade) |
